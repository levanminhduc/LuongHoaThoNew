## User's Original Request
Build a bulk payroll export feature for a Vietnamese payroll management system. Admin needs to select departments from a checklist, pick salary month and payroll type (monthly/t13), then download a single flat XLSX file containing all selected departments' payroll records in one sheet. Key decisions:
- Single flat sheet (not multi-sheet), first column = 'Phong Ban'
- POST /api/admin/bulk-payroll-export accepting {departments: string[], salary_month: string, payroll_type: 'monthly'|'t13'}
- Single Supabase DB query, group/sort in JS
- Departments with no data get a placeholder row ('Khong co du lieu')
- Shared XLSX builder extracted to lib/excel/payroll-excel-builder.ts
- Existing /api/admin/payroll-export must keep working (backward compat)
